# Productivity Suite

Este proyecto ha sido generado extrayendo los módulos de **Equipo**, **Planificación** y **Productividad (Kiosco)** de la aplicación principal.

## Instrucciones de Instalación

1.  Abre una terminal en esta carpeta:
    ```bash
    cd productivity_suite
    ```

2.  Instala las dependencias:
    ```bash
    npm install
    ```

3.  Inicia el servidor de desarrollo:
    ```bash
    npm run dev
    ```

## Módulos Incluidos

*   **Equipo**: Gestión de empleados, roles, vacaciones y bajas.
*   **Planificación**: Gestión de tareas, calendario y eventos recurrentes.
*   **Kiosco de Productividad**: Control de tiempos y lotes para el puesto de compras.
